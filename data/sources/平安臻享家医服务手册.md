### 服务达标条件
| 条件 | 说明 |
|------|------|
| 条件1 | 当年购买指定产品，单件新单总保费≥30万元，且作为投保人 |
| 条件2 | 作为有效健康险保单的被保险人 |
| 补充条件 | 若无有效健康险，需在投保指定产品当年新投保1份健康险（生效日在当年12月31日前）；若健康险中途失效，需在原服务期限内新投保1份有效健康险 |

### 权益人规则
| 规则项 | 说明 |
|--------|------|
| 主权益人 | 持有有效健康险及指定产品的客户，需同时作为有效健康险被保险人及指定产品投保人 |
| 指定权益人范围 | 主权益人的亲属（父母、配偶及其父母、子女、祖父母、外祖父母、兄弟姐妹、孙子女、外孙子女） |
| 30万≤总保费<45万 | 主权益人+2名指定权益人，其中60岁以上人员≤1名 |
| 总保费≥45万 | 主权益人+3名指定权益人，其中60岁以上人员≤2名 |
| 使用限制 | 限本人使用，不得转让 |

### 服务期限与中止/终止规则
| 规则项 | 说明 |
|--------|------|
| 服务起始 | 指定产品保险合同生效且过犹豫期后次日零时 |
| 趸交服务期限 | 1年 |
| 10年交以内（不含趸交）服务期限 | 10年 |
| 10年交及以上（含10年交）服务期限 | 保单交费期 |
| 服务中止条件1 | 健康险保单失效，服务中止；补充投保健康险生效后恢复 |
| 服务中止条件2 | 指定产品合同效力中止，服务中止；合同效力恢复后恢复 |
| 服务终止条件1 | 犹豫期退保、退保、协议解约、不如实告知、等待期出险、责任免除等导致合同解除 |
| 服务终止条件2 | 保单非有效状态或总保费不达标 |
| 服务终止条件3 | 服务期满 |
| 服务终止条件4 | 其他原因导致合同终止 |

### 重疾相关服务特殊规则（10项服务）
| 规则项 | 说明 |
|--------|------|
| 等待期 | 90天，自指定产品保单生效之日起计算 |
| 使用年龄 | 仅限70周岁(含)前使用 |
| 非共享规则 | 服务期限内，主权益人及指定权益人中仅限1人使用。一旦成功启用某一项服务，则全部10项服务将绑定至该使用人名下，其余人不可再使用 |
| 触发条件 | 保单生效90天后，首次疑似或确诊重疾（重疾目录内的重大疾病） |

### 关键合规与免责声明
1. 服务不得被理解为保险理赔结论或承诺
2. 第三方服务供应商仅提供协助预约，不参与提供医疗服务
3. 就医参考信息仅供参考，客户拥有最终决定权
4. 客户应寻求并依赖医疗机构及医生的专业判断
5. 就医产生的所有医疗费用由客户自行承担
6. 使用服务前需完成信息授权
7. 平安人寿有权调整服务内容，并会通知客户
8. 名医大咖、数字化管理的智能设备需客户自行付费购买

常见问题

Q1  
多份符合约定的保险合同的服务权益如何计算？  
按照符合约定的指定产品保险合同的份数，确定服务权益份数。

Q2  
什么时候可享受服务权益？  
所有服务均是对应的指定产品保险合同生效且过犹豫期后次日零时起即可  
使用（对于含有启动条件的服务需同时满足启动条件）。

Q3  
为什么在平安金管家APP看不到服务权益？  
1、请确保您的保单在约定保险产品范围内，且处于有效交费状态；  
2、请确认您登陆平安金管家APP的手机号与在平安人寿留存的主权  
益人手机号一致；3、请确认指定产品保单的投保人拥有有效健康险  
保单，且同样作为健康险保单的被保人。

Q4  
服务权益谁可以使用？  
若满足30万≤指定产品单件新单总保费<45万，则限主权益人和2名  
指定权益人领取使用，主权益人及其指定权益人中仅可包含1名超过  
60岁的人员；若满足45万≤指定产品单件新单总保费，则限主权益  
人和3名指定权益人领取使用，主权益人及其指定权益人中仅可包含2  
名超过60岁的人员；权益激活时，主权益人可在其亲属（父母、配偶  
及其父母、子女、祖父母、外祖父母、兄弟姐妹、孙子女、外孙子  
女）中选择相应人员作为指定权益人。  
（部分重疾相关服务含有使用条件，详见本服务手册中服务体系、服  
务内容及标准部分相关内容）

Q5  
服务权益提供多久？  
详见“服务期限”部分内容。

Q6  
服务必须达标当年激活使用吗？  
当年度达标后，只要指定产品和健康险持续有效且满足保费要求，后  
续服务期限内任一年激活使用都可以。

若您对服务有任何不满，可拨打7*24小时人工维权服务热线4001666333。
