# 臻享家医服务计划

## 产品定位

臻享家医服务计划是一款面向指定产品高保费客户的增值服务权益包，通过设置保费门槛（30-45万）和两个服务等级（V1、V2优享），为客户提供18项健康管理和就医服务，作为产品的差异化卖点。

## 基础规则

| 项目 | 内容 |
|:---|:---|
| 服务品牌 | 臻享家医 |
| 服务等级 | V1、V2优享 |
| V1等级达标条件 | 指定产品单件新单总保费 ≥ 30万 且 < 45万 |
| V1等级可享权益数 | 18项 |
| V2优享达标条件 | 未在源文档中显示，需查阅服务手册 |
| 指定产品列表 | 未在源文档中说明，需与业务方确认 |
| 使用人数 | 未在源文档中说明，以服务手册为准 |
| 服务有效期 | 未在源文档中说明，需查阅服务手册 |

## 核心权益/服务

臻享家医服务计划V1等级包含以下18项服务权益：

| 序号 | 服务名称 | 服务类别 |
|:---:|:---|:---|
| 1 | 特色体检 | 健康管理/体检 |
| 2 | 住院安排协助 | 就医服务/住院 |
| 3 | 海外就医协助 | 就医服务/海外 |
| 4 | 门诊预约协助及陪诊 | 就医服务/门诊 |
| 5 | 音视频问诊 | 健康管理/在线问诊 |
| 6 | 慢病预防 | 健康管理/慢病与重疾 |
| 7 | 重疾专案管理 | 健康管理/慢病与重疾 |
| 8 | 家庭医生 | 健康管理/在线问诊 |
| 9 | 心理咨询 | 健康管理/心理健康 |
| 10 | 检查安排协助 | 就医服务/检查 |
| 11 | 专家会诊 | 就医服务/会诊 |
| 12 | 手术安排协助 | 就医服务/手术 |
| 13 | 住院照护 | 就医服务/住院 |
| 14 | 出院安排协助 | 就医服务/出院 |
| 15 | 康复门诊协助 | 就医服务/康复 |
| 16 | 康复住院协助 | 就医服务/康复 |
| 17 | 上门护理 | 就医服务/护理 |
| 18 | 在线问诊 | 健康管理/在线问诊 |

## 适配客户

- 购买指定产品且单件新单总保费达到30万及以上的客户
- 对健康管理和就医服务有较高需求的客户
- 注重服务体验和便捷性的中高端客户

## 销售方法关联

- 可作为高保费产品的增值卖点，提升产品吸引力
- 可用于异议处理中回应客户关于服务价值的疑问
- 18项服务权益清单可作为产品讲解时的亮点展示
