# 平保寿发〔2025〕561号附件3：平安盛世优享（2026）养老年金保险险种投保规则

## 原文事实清单

本文件为《平安盛世优享（2026）养老年金保险》（险种简称：盛世优享26，险种代码：1822）的投保规则清单，共14条。未涉及之规定以《个人寿险核保规则(2025年)》规定为准。

| 序号 | 规则项 | 具体内容 |
| :--- | :--- | :--- |
| 1 | 销售渠道 | 个险（A01） |
| 2 | 投保年龄 | 18-64岁 |
| 3 | 交费期 | 趸交、至50/55/60/65岁、6/10/15/20年 |
| 4 | 交费方式 | 年交、趸交 |
| 5 | 保险期间 | 至领取期间届满时止 |
| 6 | 领取规则 | **开始领取年龄**：男性可选60-70岁期间保单周年日起领；女性可选55-70岁期间保单周年日起领<br>**领取方式及期间**：趸交时可以选择年领；年交时可以选择一次性领取或年领；年领时，领取期间为20年 |
| 7 | 投、被保险人关系 | 本人 |
| 8 | 险种组合规则 | 需单独投保盛世优享26 |
| 9 | 最低保额 | 不受传统寿险保额不低于10000元的限制；保费算保额时无需控制保额为100元整数倍；保额算保费时保额需控制为1000元整数倍 |
| 10 | 起保点 | **趸交**：最低1万保费起<br>**年交**：最低年交保费2000元起<br>**同一客户投保盛世优享系列最高保费累计不超过12000元** |
| 11 | 风险保额计算 | 不计风险保额 |
| 12 | 职业加费 | 无职业加费 |
| 13 | 健康告知规则 | 被保险人无健康询问 |
| 14 | 核保规则 | **自核通过件**：标体承保<br>**自核不通过件**：送人工核保，不审核健康风险因素<br>**达到法律法规及监管要求的情况**（如高风险、黑名单客户、反洗钱标准等）：送人工核保 |

## 原始清单明细
