# 平保寿发〔2025〕561号附件2：平安盛世优享（2026）养老年金保险费率表

### 费率表结构

费率表按以下维度分组：

1. **性别**：男性、女性（费率表分开列示）
2. **开始领取年龄**：
   - 男性：60岁、61岁、62岁、63岁、64岁、65岁、66岁、67岁、68岁、69岁、70岁
   - 女性：55岁、56岁、57岁、58岁、59岁、60岁、61岁、62岁、63岁、64岁、65岁、66岁、67岁、68岁、69岁、70岁
3. **缴费方式**：趸交、6年交、10年交、15年交、20年交、交至50岁、交至55岁、交至60岁、交至65岁
4. **领取方式**：一次性领取、领取期间=20年

### 费率数值含义

表格中的数值代表**每万元趸交或年交保费对应的基本保险金额**。例如，男性18岁，趸交，60岁开始领取，基本保险金额为122,172元，意味着投保人一次性缴纳10,000元保费，在60岁开始领取时，可获得122,172元的保险金额。

### 数据规律

| 规律 | 说明 | 示例 |
|------|------|------|
| 年龄效应 | 投保年龄越小，基本保险金额越高 | 男性60岁领取趸交：18岁122,172元 → 30岁96,018元 → 40岁78,486元 |
| 缴费期效应 | 缴费期限越长，基本保险金额越高 | 男性18岁60岁领取：趸交122,172元 → 6年交197,520元 → 10年交282,308元 → 20年交527,561元 |
| 领取年龄效应 | 开始领取年龄越大，基本保险金额越高 | 男性18岁趸交：60岁122,172元 → 61岁124,743元 → 62岁127,380元 |
| 性别效应 | 相同条件下女性基本保险金额通常高于男性 | 女性平均寿命更长，费率设计体现性别差异 |
