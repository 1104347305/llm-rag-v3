# 平保寿发〔2025〕559号附件3：平安智盈倍护（2026）终身护理保险A01投保规则

## 原文事实清单

本文件为《平安智盈倍护（2026）终身护理保险》A01投保规则，共包含13条核心规则及附件一险种基本信息一览表。以下为逐条事实清单：

| 编号 | 规则项 | 具体内容 |
| :--- | :--- | :--- |
| 1 | 销售渠道 | 个险（A01） |
| 2 | 投保年龄 | 6年交：0-64周岁；10/15年交：0-60周岁；20年交：0-55周岁；交至55岁：0-49周岁；交至60岁：0-54周岁；交至65岁：0-59周岁 |
| 3 | 投被保险人关系 | 本人、配偶、父母、子女 |
| 4 | 交费期 | 6/10/15/20年交，交至55/60/65岁 |
| 5 | 交费方式 | 年交 |
| 6 | 保险期间 | 终身 |
| 7 | 起保点 | 保费最低1200元 |
| 8 | 保额保费规则 | 最低保额不受传统寿险保额不低于10000元的限制；保费算保额，保费控制为100元整数倍；保额算保费，保额需控制为1000元整数倍 |
| 9 | 险种组合规则 | 智盈倍护26单独投保 |
| 10 | 风险保额计算 | 承保后计算残护责任保额。18周岁以下：不计；18-60周岁：总交保费*60%；61周岁及以上：1倍基本保额。**注**：需与御享照护护理(1663)、臻享照护护理(1566)、臻宝照护护理(1568)、长护保(1237)、智盈倍护(2635)、智盈倍护25(2642)及智盈倍护26(2648)累计 |
| 11 | 健康告知 | 单独投保健康告知，包含3大类问题：<br>1) 目前或曾患疾病（高血压2级及以上、心脏/血管疾病、颅脑/神经疾病、其他疾病如肢体缺失、瘫痪等）；<br>2) 过去一年内症状（运动迟缓、肢体震颤、肌肉萎缩、复视、视力下降、记忆力/判断力减退）；<br>3) 过去两年内被建议住院或手术（脊髓疾病、脊柱和脊髓损伤、脊柱结核、脊髓灰质炎、重症肌无力、严重的颅脑外伤） |
| 12 | 职业规则 | 寿险1-6类职业，无职业加费 |
| 13 | 核保规则 | 参考重疾审核，EM≤150标体，EM＞150拒保，不做弱体加费，原则上不接受除外责任 |
