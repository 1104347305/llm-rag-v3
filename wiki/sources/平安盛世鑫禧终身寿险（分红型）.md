# 平安盛世鑫禧终身寿险（分红型）条款

## 原文事实清单

本文件为《平安盛世鑫禧终身寿险（分红型）》正式保险条款，共16页，由中国平安人寿保险股份有限公司发布，监管备案号平安人寿〔2025〕终身寿险051号。

### 产品基础信息

| 字段 | 值 |
|:---|:---|
| 险种简称 | 盛世鑫禧 |
| 险种代码 | 3188 |
| 产品全称 | 平安盛世鑫禧终身寿险（分红型） |
| 监管备案号 | 平安人寿〔2025〕终身寿险051号 |
| 产品类型 | 分红型终身寿险 |
| 投保年龄 | 0周岁（须出生满28日）至60周岁 |
| 保险期间 | 终身 |
| 犹豫期 | 20日 |
| 交费方式 | 月交、季交、半年交、年交（由条款推断） |
| 宽限期 | 60日 |

### 核心保险责任

| 责任名称 | 触发条件 | 给付金额 | 备注 |
|:---|:---|:---|:---|
| 身故保险金 | 被保险人身故 | 见下方详细规则 | 与意外全残保险金仅给付一项 |
| 意外全残保险金 | 意外伤害导致全残（180日内） | 见下方详细规则 | 与身故保险金仅给付一项 |

#### 身故保险金详细规则

**18周岁前（不含）身故：** max(所交保险费, 现金价值)

**18周岁后（含）身故且交费期间未届满：** max(所交保险费×比例, 月交保险费×12×应交费年度数, 现金价值)

**18周岁后（含）身故且交费期间已届满：** max(所交保险费×比例, 当年保障额度, 现金价值)

#### 意外全残保险金详细规则

结构与身故保险金类似，触发条件为意外伤害导致全残（180日内）。

#### 身故/全残给付比例表

| 到达年龄 | 比例 |
|:---|:---|
| 小于40周岁（含） | 160% |
| 41周岁（含）- 60周岁（含） | 140% |
| 61周岁（含）及以上 | 120% |

#### 当年保障额度计算公式

当年保障额度 = 基本保险金额 × (1 + 2.0%)^(n-1)，其中n为保单年度数。

### 保单红利

- 红利是不保证的
- 红利领取方式：累积生息、抵交保险费、购买交清增额保险
- 未选择时默认累积生息
- 红利分配于保单周年日

### 责任免除（身故）

1. 投保人对被保险人的故意杀害、故意伤害
2. 被保险人故意犯罪或抗拒刑事强制措施
3. 本合同成立或复效之日起2年内自杀（无民事行为能力人除外）
4. 吸毒
5. 酒后驾驶
6. 战争、军事冲突、暴乱或武装叛乱
7. 核爆炸、核辐射或核污染

### 责任免除（意外全残）

在身故免责基础上增加：
8. 妊娠、流产、分娩导致的伤害
9. 药物过敏或精神和行为障碍（ICD-10）
10. 未遵医嘱私自使用药物（非处方药除外）
11. 高风险运动（潜水、跳伞、攀岩、蹦极、滑翔、探险、摔跤、武术比赛、特技表演、赛马、赛车等）

### 其他重要规则

| 规则 | 内容 |
|:---|:---|
| 基本保额减少限制 | 同一保单年度内累计减少不超过生效时基本保额的20% |
| 保险事故通知 | 知道后10日内通知 |
| 保险金核定 | 5日内核定，情形复杂30日内 |
| 保单贷款 | 不超过现金价值80%扣除欠款，每次最长6个月 |
| 自动垫交 | 可用现金价值垫交欠交保费 |
| 合同效力中止 | 宽限期后未支付保费，或贷款本息达现金价值 |
| 合同效力恢复 | 中止后2年内可申请复效 |
| 年龄错误处理 | 真实年龄不符可解除合同；少交保费补交并按比例给付；多交保费退还 |
| 未成年人身故限制 | 符合监管规定 |
| 合同解除权限制 | 自知悉解除事由起30日内行使，合同成立超过2年不得解除 |

### 全残项目表（附表）

条款附表详细列出了60余种全残情形及判定标准，包括：
- 颅脑损伤导致智力缺损（3级）、植物状态
- 眼球缺失、视力/视野缺损（5级标准）
- 听力损失、耳廓缺失
- 舌缺损
- 胸部/腹部损伤导致器官切除或移植
- 会阴部损伤导致睾丸缺失/萎缩
- 颌骨缺失、牙齿脱落
- 肢体缺失/功能丧失（多种组合）
- 瘫痪（四肢瘫、偏瘫、截瘫，肌力分级0-5级）
- 烧伤（III度，面积按中国新九分法计算）
- 皮肤瘢痕（面积计算标准）
