# 平安康养会员代理人指导手册-入会服务篇（23年9月版）

## 原文事实清单

### 1. 核心规则

| 类别 | 事实描述 |
| :--- | :--- |
| **会员层级** | 逸享会员、逸享PLUS会员、颐享家会员、臻享会员V1-V3 |
| **服务期限** | 与保单有效期保持一致，最长可达终身 |
| **服务启动** | 入会即启用健康服务（健康+/七维健康）和居家健康服务（逸享会员无）；女性满55周岁/男性满60周岁启用居家管家服务（逸享会员无）；79岁起启用臻颐年特色服务（仅颐享家/臻享会员） |
| **服务性质** | 康养会员服务并非保险产品及相关服务，由平安集团旗下其他子公司及/或第三方服务机构提供 |
| **积分规则** | 会员级别越高，积分越多。积分可用于兑换七维健康服务或臻颐年特色服务。积分有效期与会员资格有效期一致 |
| **资格转让** | 会员资格仅限投保人本人享有，不可转让 |
| **资格失效** | 当客户名下无有效保单时，会员资格失效，积分冻结，所有免费或赠送次数的服务将无法使用 |
| **免费权益有效期** | 免费权益次数有效期为当年1月1日至12月31日（含），跨年不可累加 |
| **服务入口** | 平安金管家APP（我的>康养会员）；臻享会员专线4008895511（咨询/预约）；居家服务入口：4008795511电话、智能音箱、居家养老管家企业微信 |

### 6. 关键服务细节

- **家庭医生/音视频问诊**：7*24小时服务，60秒内响应。首次使用需签约家庭医生。
- **私人教练/营养师**：服务时间9:00-21:00，60秒内响应，24h内生成定制方案（节假日顺延）。
- **特色体检**：覆盖246城市573网点，12h内确认预约。提供18项专项深度检查。
- **院后护理**：提交申请后36小时内完成预约确认。
- **重疾专案管理**：海外远程书面咨询及海外重疾住院安排协助为自费服务。
- **慢病防控**：1个工作日内进行电话首访和完成管理方案制定。服务团队包括医师（≥10年）、健康管理师（≥3年）、营养师（≥5年）、心理咨询师（≥3年）。
- **七维健康服务**：多项服务需积分兑换，服务地点为深圳平安粤海门诊部，服务时间多为每周二至周六8:00~16:00。

| 居家养老服务一直提供吗？ | 与康养会员资格挂钩，保单有效期内持续提供 |
| 康养会员级别如何查询？ | 登录平安金管家APP或致电会员专线4008895511 |
