# 平安附加e生保（尊享版）长期医疗保险（费率可调）

## 产品定位

平安附加e生保（尊享版）长期医疗保险（费率可调）是平安人寿推出的一款**20年保证续保**的长期医疗保险产品。该产品定位为"长期保障、责任全面、费率调整公开透明"，旨在为客户提供长期稳定的医疗保障，解决客户对医疗费用风险的担忧。

## 基础规则

| 规则项 | 内容 |
| :--- | :--- |
| 产品类型 | 费率可调的长期医疗保险 |
| 保证续保期间 | 20年 |
| 上市销售时间 | 2022年4月20日 |
| 首次费率调整时间 | 不早于2025年4月20日 |
| 费率调整间隔 | 每次不少于1年 |
| 单次费率调整上限 | 30% |
| 费率调整公示 | 官网"公开信息披露"专栏，公示期不短于30日 |
| 续保规则 | 保证续保期间内，若未收到不再续保的书面通知，视作申请续保 |

## 核心保障

### 一般医疗保险金（年度限额200万）

- 住院医疗费用
- 指定门诊医疗费用
- 住院前后门诊急诊费用

### 重大疾病医疗保险金（年度限额200万）

- 120种重大疾病住院医疗费用
- 120种重大疾病指定门诊医疗费用
- 120种重大疾病住院前后门诊急诊费用

### 其他关键参数

- **保证续保期间内总给付限额**：800万
- **年免赔额**：1万
- **给付比例**：100%（若投保时有社保但就诊时未使用，则为60%）
- **无理赔优惠**：限额可根据无理赔情况增加（详见条款1.2.6）

## 责任免除

本产品包含25项责任免除条款，涵盖既往症、遗传性疾病、美容、矫形、生育、高风险运动、战争、实验性治疗、基因疗法和细胞免疫疗法等。完整清单见[[平安附加e生保（尊享版）长期医疗保险（费率可调）]]的原文事实清单。

## 费率调整规则

本产品为费率可调的长期医疗保险，费率调整规则详见[[平安附加e生保（尊享版）费率调整规则]]。

## 适配客户

本产品适合以下客户群体：
- 希望获得长期医疗保障的客户
- 关注医疗费用风险，希望转移大额医疗支出风险的客户
- 有社保但希望获得更全面保障的客户
- 年龄在投保范围内的客户（具体年龄范围待补全）

## 合规提醒

- 本产品为费率可调的长期医疗保险，费率可能调整，需向客户明确说明
- 费率调整规则公开透明，需向客户解释调整触发条件、上限和流程
- 投保人不同意调费可申请不再续保，需告知客户此项权利
- 投保举例仅为演示，不代表对未来调整费率的预期，需避免误导客户
