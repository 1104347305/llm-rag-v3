# 智盈倍护26_保额保费规则

## 规则内容

| 规则项 | 内容 |
| :--- | :--- |
| **最低保额限制** | 不受传统寿险保额不低于10000元的限制 |
| **保费算保额** | 保费控制为100元整数倍 |
| **保额算保费** | 保额需控制为1000元整数倍 |

## 销售意义

本产品的投保门槛较低，最低保费仅1200元起，且最低保额不受传统寿险1万元限制，适合预算有限的客户或作为补充保障。

## 来源依据

本规则来源于《平保寿发〔2025〕559号附件3：平安智盈倍护（2026）终身护理保险A01投保规则》第8条。
