# 平安少儿盛世福（2023）重大疾病保险

## 产品定位
平安少儿盛世福（2023）重大疾病保险是中国平安人寿保险股份有限公司推出的少儿专属终身重大疾病保险产品。该产品专为0-17周岁未成年人设计，提供覆盖120种重大疾病、40种轻度疾病、20种中症疾病和15种少儿特定重疾的全面保障，并可选配重大疾病特别保险金和陪护保险金，满足少儿成长过程中的健康保障需求。

## 基础规则
| 项目 | 内容 |
| :--- | :--- |
| 投保年龄 | 0周岁至17周岁（0周岁须为出生满28日且已健康出院的婴儿） |
| 等待期 | 90日（意外伤害无等待期） |
| 保障期间 | 基本部分：终身；可选A部分：约定载明；可选B部分：至25周岁保单周年日 |
| 犹豫期 | 20日 |
| 宽限期 | 60日 |
| 保单贷款 | 不超过现金价值80%，最长6个月 |
| 减额交清 | 仅基本部分可办理 |

## 核心保障/权益

### 基本部分
1. **身故保险金**：18周岁前返还已交保费，18周岁后赔付基本保险金额。
2. **轻度疾病保险金**：确诊40种轻度疾病之一，赔付基本保险金额的20%，累计最多6次，每种限1次。
3. **中症疾病保险金**：确诊20种中症疾病之一，赔付基本保险金额的50%，限1次。
4. **少儿特定重疾保险金**：确诊15种少儿特定重疾之一，赔付基本保险金额的100%，限1次。
5. **重大疾病保险金**：确诊120种重大疾病之一，赔付基本保险金额的100%，限1次。给付后基本部分责任终止。

### 可选部分A
- **重大疾病特别保险金**：确诊重大疾病后，额外赔付基本保险金额的50%，限1次。

### 可选部分B（陪护保险金）
- 按份销售，每份每月2000元。
- **轻度疾病陪护保险金**：每月给付，连续3个月，累计最多6次。
- **中症疾病陪护保险金**：每月给付，连续3个月，限1次。
- **少儿特定重疾陪护保险金**：每月给付，连续10个月，限1次。
- **重大疾病陪护保险金**：每月给付，连续10个月，限1次。
- 若被保险人身故，剩余月数一次性给付继承人。
- 所有陪护保险金责任在25周岁保单周年日零时终止。

### 70周岁前额外给付规则
- 每发生一次轻度疾病，额外身故/少儿特定/重大疾病保险金比例增加10%，最高60%。
- 发生一次中症疾病，额外身故/少儿特定/重大疾病保险金比例为20%。

## 疾病种类
| 类别 | 数量 | 行业规范定义 | 说明 |
| :--- | :--- | :--- | :--- |
| 轻度疾病 | 40种 | 3种 | 包括恶性肿瘤—轻度、原位癌、较轻急性心肌梗死等 |
| 中症疾病 | 20种 | 0种 | 包括早期原发性心肌病、中度脑损伤等 |
| 少儿特定重疾 | 15种 | 0种 | 包括白血病、严重癫痫、重症手足口病等 |
| 重大疾病 | 120种 | 28种 | 包括恶性肿瘤—重度、较重急性心肌梗死、冠状动脉搭桥术等 |

## 责任免除
包含9类标准免责情形：投保人故意杀害/伤害、被保险人故意犯罪、2年内自杀（无民事行为能力人除外）、吸毒、酒后驾驶、感染艾滋病病毒（特定情形除外）、战争/军事冲突、核爆炸/辐射/污染、遗传性疾病/先天性畸形/染色体异常。

## 适配客户
- **目标客户**：0-17周岁未成年人的父母或监护人。
- **核心需求**：为孩子提供终身重大疾病保障，覆盖少儿高发特定重疾，同时通过陪护保险金弥补家长陪护期间的收入损失。
- **购买信号**：关注孩子健康、有家族病史、希望给孩子全面保障的父母。

## 合规提醒
- 销售时必须明确告知等待期、责任免除、犹豫期等关键条款。

- 重大疾病保险金和身故保险金仅给付其中一项。
- 可选B部分陪护保险金在25周岁保单周年日零时终止。
