# 个人养老金投保限制规则_盛世优享26

## 规则内容

按个人养老金制度投保平安盛世优享（2026）养老年金保险的客户，适用以下特殊限制：

### 不适用权益

| 权益 | 限制内容 | 条款依据 |
| :--- | :--- | :--- |
| **保单贷款** | 不接受保单贷款申请 | 条款6.2 |
| **自动垫交** | 不接受自动垫交申请 | 条款6.3 |

### 特殊要求

| 要求 | 内容 | 条款依据 |
| :--- | :--- | :--- |
| **保险费支付** | 保险费需从个人养老金资金账户支付 | 条款7.8 |
| **资金往来** | 基于本合同的相关资金往来应按《个人养老金实施办法》等规定执行 | 条款7.8 |

## 风险类型

服务承诺/权益限制

## 适用对象

- [[平安盛世优享（2026）养老年金保险]] — 按个人养老金制度投保的客户

## 必要声明

需在销售时明确告知个人养老金客户以下限制：
1. 本产品为个人养老金专属产品，保险费必须从个人养老金资金账户支付
2. 个人养老金投保人不享有保单贷款权益
3. 个人养老金投保人不享有自动垫交权益
4. 减额交清权益仍然适用

## 来源依据

- 条款6.2：保单贷款 — "若您按照个人养老金制度投保并订立本合同的，我们不接受保单贷款申请。"
- 条款6.3：自动垫交 — "若您按照个人养老金制度投保并订立本合同的，我们不接受自动垫交申请。"
- 条款7.8：特别约定 — "若您按照国家关于个人养老金的法律、行政法规或其他相关规定、政策投保并订立本合同的，保险费需从个人养老金资金账户支付。"
