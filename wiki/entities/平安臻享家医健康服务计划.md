# 平安臻享家医健康服务计划

## 产品定位

平安臻享家医健康服务计划是平安人寿面向高净值客户提供的家庭医生健康服务权益包，以"1个专业家庭医生团队+1个标准路径+3套主动健康管理服务+N项优质医疗资源"为核心模式，为客户打造"省心、省时又省钱"的一站式健康服务体验。

## 基础规则

- **版本**: 2025年4月版
- **适用保单生效日期**: 2025年4月21日(含)及以后
- **服务提供方**: 平安健康互联网股份有限公司及其上海分公司、北京北医医院管理有限公司
- **服务入口**: 平安金管家APP
- **客服热线**: 95511
- **投诉热线**: 4001666333

### 服务达标条件
| 条件 | 说明 |
|------|------|
| 条件1 | 当年购买指定产品，单件新单总保费≥30万元，且作为投保人 |
| 条件2 | 作为有效健康险保单的被保险人 |
| 补充条件 | 若无有效健康险，需在投保指定产品当年新投保1份健康险（生效日在当年12月31日前）；若健康险中途失效，需在原服务期限内新投保1份有效健康险 |

### 权益人规则
| 规则项 | 说明 |
|--------|------|
| 主权益人 | 持有有效健康险及指定产品的客户，需同时作为有效健康险被保险人及指定产品投保人 |
| 指定权益人范围 | 主权益人的亲属（父母、配偶及其父母、子女、祖父母、外祖父母、兄弟姐妹、孙子女、外孙子女） |
| 30万≤总保费<45万 | 主权益人+2名指定权益人，其中60岁以上人员≤1名 |
| 总保费≥45万 | 主权益人+3名指定权益人，其中60岁以上人员≤2名 |
| 使用限制 | 限本人使用，不得转让 |

### 服务期限
| 交费方式 | 服务期限 |
|----------|----------|
| 趸交 | 1年 |
| 10年交以内（不含趸交） | 10年 |
| 10年交及以上（含10年交） | 保单交费期 |

## 适配客户

- **目标客户**: 单件新单总保费≥30万的高净值家庭客户
- **客户特征**: 有健康险需求，关注家庭健康管理，愿意为全家健康投入
- **购买信号**: 咨询家庭健康保障方案、关注高端医疗服务、有家庭成员健康管理需求

## 合规提醒

- 服务不得被理解为保险理赔结论或承诺
- 第三方服务供应商仅提供协助预约，不参与提供医疗服务
- 就医参考信息仅供参考，客户拥有最终决定权
- 客户应寻求并依赖医疗机构及医生的专业判断
- 就医产生的所有医疗费用由客户自行承担
- 使用服务前需完成信息授权
- 平安人寿有权调整服务内容，并会通知客户
- 名医大咖、数字化管理的智能设备需客户自行付费购买
