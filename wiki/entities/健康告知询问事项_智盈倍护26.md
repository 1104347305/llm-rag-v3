# 健康告知询问事项_智盈倍护26

## 规则概述

本规则规定了投保《平安智盈倍护（2026）终身护理保险》时必须进行的健康告知询问事项。这些询问事项是核保决策的核心依据，也是未来理赔时判断是否如实告知的重要参考。

## 询问事项内容

### 询问事项1：既往病史

被保人目前或曾经是否患有下列疾病或症状？

| 类别 | 具体疾病/症状 |
| :--- | :--- |
| **血压血糖** | 高血压病2级及以上（收缩压≥160mmHg或舒张压≥100mmHg），糖尿病或空腹血糖≥7.0mmo/L |
| **心脏、血管疾病** | 心房颤动，心房扑动，感染性心内膜炎，风湿性心脏病，心肌梗死，肺动脉高压 |
| **颅脑、神经疾病** | 脑血管畸形，脑血管瘤，颅内肿瘤、囊肿或占位，脊髓肿瘤或占位，脑炎（且目前存在后遗症），脑膜炎（且目前存在后遗症），脑梗死，脑出血，阿尔茨海默病，帕金森病，运动神经元病，多发性硬化症、肌萎缩侧索硬化 |
| **其他疾病** | 一个及以上肢体腕关节或踝关节以上缺失，肢体功能障碍，瘫痪，肌营养不良症，青光眼，视神经病变，黄斑变性，单眼失明，双眼失明，智力障碍 |

### 询问事项2：近期症状

过去一年内是否曾有下列症状？
- 运动迟缓
- 肢体震颤
- 原因不明的肌肉萎缩
- 复视
- 视力明显下降
- 记忆力或判断力进行性减退

### 询问事项3：近期手术建议

被保险人过去两年内是否因下列疾病而被医生建议住院或手术？
- 脊髓疾病
- 脊柱和脊髓损伤
- 脊柱结核
- 脊髓灰质炎
- 重症肌无力
- 严重的颅脑外伤

## 合规提醒

- 健康告知是投保人的法定义务，需如实告知
- 未如实告知可能导致保险合同解除或理赔拒赔
- 询问事项覆盖了与护理保险责任高度相关的疾病和症状（如帕金森病、阿尔茨海默病、运动神经元病等）
- 代理人应引导客户逐项阅读并如实回答
