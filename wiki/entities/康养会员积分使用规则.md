# 康养会员积分使用规则

## 规则定义
康养会员积分是平安集团根据会员级别配给的专享积分，客户可使用积分自由兑换本材料中提及的各项服务。积分仅限用于兑换服务，不得用于其他用途。

## 积分配给标准
| 会员级别 | 累计可获积分 |
|----------|--------------|
| 逸享会员 | 588万及以上积分 |
| 逸享PLUS会员 | 1000万及以上积分 |
| 颐享家会员 | 688万及以上积分 |
| 臻享会员V1 | 1888万及以上积分 |
| 臻享会员V2 | 3288万及以上积分 |
| 臻享会员V3 | 4388万及以上积分 |

## 使用规则
| 项目 | 内容 |
|------|------|
| 使用范围 | 仅适用于兑换本材料中提及的各项服务，不做他用 |
| 兑换对象 | 七维健康服务、臻颐年特色服务等 |
| 积分性质 | 自由兑换，客户可根据需求选择服务 |
| 积分查询 | 平安金管家APP > 我的 > 康养会员 > 查看积分余量及消耗历史 |

## 影响范围
- 适用对象：所有康养会员（逸享、逸享PLUS、颐享家、臻享V1-V3）。
- 积分有效期：与保单有效期保持一致。

## 销售意义
- 积分体系是康养会员差异化权益的重要体现，级别越高积分越多，激励客户升级。
- 积分兑换服务（如七维健康、臻颐年特色）可满足高端客户的个性化需求。
- 代理人可向客户展示积分余量及可兑换服务，增强会员价值感。

## 风险提醒
- **非保险产品**：康养会员服务并非保险产品及相关服务，客户签署的保险合同由签约保险公司履约；康养会员服务由平安集团旗下其他子公司及第三方服务机构提供。
- **积分用途限制**：积分仅限兑换本材料提及的服务，不得兑换现金或其他非服务项目。
- **积分调整**：积分配给标准可能根据市场状况、会员需求和公司规划等因素调整，代理人不得承诺固定积分值。

## 知识差距
- 积分兑换的具体操作流程（如兑换后是否可撤销、兑换时效）未详细说明。
- 积分是否可累积、是否可转让未明确。
- 积分过期规则（如保单失效后积分是否清零）未说明。
