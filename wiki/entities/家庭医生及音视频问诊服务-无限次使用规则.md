# 家庭医生及音视频问诊服务无限次使用规则

## 规则定义
康养会员在服务有效期内，可全年无限次免费使用家庭医生及音视频问诊服务。该服务通过签约专属家庭医生，建立健康档案，提供专业医疗诊断、就医服务、客户权益管理及覆盖健康医疗场景的主动管理式服务；音视频问诊则通过实时或预约方式与医生面对面沟通，提升服务体验。

## 触发条件
- 客户须为康养会员（逸享会员、逸享PLUS会员、颐享家会员、臻享会员V1-V3均可享有）。
- 首次使用前，需选择一位符合要求的家庭医生进行签约并激活家庭医生服务。
- 服务有效期与保单有效期保持一致。

## 使用规则
| 项目 | 内容 |
|------|------|
| 服务次数 | 全年无限次免费使用 |
| 首次使用 | 需选择一位家庭医生签约并激活 |
| 服务团队 | 平安健康医生团队；家庭医生为主治级医生；音视频问诊、在线问诊医生为主治及以上为主 |
| 服务时间 | 7×24小时（音视频问诊时段为9:00-23:00） |
| 响应时效 | 提出咨询需求后60秒内回复 |
| 服务入口 | 平安金管家APP > 我的 > 康养会员 > 健康服务页面 |

## 影响范围
- 适用对象：所有康养会员级别（逸享、逸享PLUS、颐享家、臻享V1-V3）。
- 服务覆盖：健康管理全生命周期，包括日常健康咨询、常见病问诊、就医指导等。

## 销售意义
- 无限次使用是康养会员的核心卖点之一，可向客户强调“全年不限次、随时问诊”的便利性。
- 首次签约家庭医生的流程简单，有助于建立长期客户粘性。
- 7×24小时服务及60秒响应时效，体现高端服务品质。

## 风险提醒
- 服务由平安集团旗下其他子公司及第三方服务机构提供，非保险产品本身。
- 最终服务内容以金管家APP实际展示为准，代理人不得做出超出材料范围的承诺。
- 音视频问诊服务时间限制（9:00-23:00）需向客户明确说明，避免夜间问诊误解。

## 知识差距
- 家庭医生签约后是否可以更换？更换频率和流程未明确。
- 无限次使用是否存在合理使用限制（如滥用防范机制）？
- 音视频问诊在非服务时段（23:00-9:00）是否提供其他替代服务？
