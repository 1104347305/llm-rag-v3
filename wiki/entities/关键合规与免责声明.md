# 关键合规与免责声明

## 规则定义

本合规与免责声明是平安臻享家医健康服务计划的核心法律文件，包含8项具体条款，全面覆盖服务性质界定、第三方责任划分、就医信息使用、医疗费用承担、信息授权要求及服务内容调整权等关键法律事项。该声明适用于所有29项服务项目，是服务提供方与客户之间的法律基础。

## 触发条件

本声明在以下场景中自动生效：
- 客户首次了解服务计划时
- 客户激活服务前
- 客户使用任何一项服务时
- 服务内容发生调整时
- 客户与第三方服务供应商产生纠纷时

## 影响范围

| 受影响方 | 具体影响 |
|----------|----------|
| 客户 | 需自行承担医疗费用；就医决策需依赖专业医疗判断；服务前需完成信息授权 |
| 销售人员 | 不得将服务承诺为保险理赔结论；需向客户明确服务的非保险性质 |
| 服务提供方 | 仅提供协助预约和就医参考信息，不参与提供医疗服务 |
| 第三方服务供应商 | 仅提供预约协助，不参与提供医疗服务 |

## 销售意义

1. **合规销售基础**：销售人员必须将本声明作为销售话术的组成部分，避免误导客户将服务等同于保险理赔承诺。
2. **客户预期管理**：明确告知客户服务仅为协助和参考性质，医疗费用需自理，避免后续纠纷。
3. **风险隔离**：通过明确第三方服务供应商的职责边界，降低平安人寿的法律风险。
4. **信息授权必要性**：客户需完成信息授权后方可使用服务，这是服务启动的前置条件。

## 风险提醒

- **禁止承诺**：严禁销售人员将服务内容承诺为保险理赔结论或保证性医疗结果。
- **费用说明**：名医大咖服务和数字化管理的智能设备需客户自行付费购买，销售人员需明确告知。
- **服务调整权**：平安人寿有权调整服务内容，销售人员不得承诺服务内容永久不变。
- **医疗决策权**：客户拥有最终决定权，销售人员不得代替客户做出医疗决策。

## 条款详解

### 条款1：服务不得被理解为保险理赔结论或承诺

本服务计划为健康管理增值服务，非保险产品。服务提供方不承诺任何医疗结果或保险理赔结论。客户不得将服务内容作为保险理赔的依据。

**合规要点**：
- 销售话术中不得出现"保证治愈""包治百病"等绝对化用语
- 不得将服务与保险理赔挂钩
- 服务手册中需明确标注"服务非保险理赔承诺"

### 条款2：第三方服务供应商仅提供协助预约

第三方服务供应商（如医院、体检机构）仅提供预约协助服务，不参与提供医疗服务。客户与第三方之间的任何纠纷，平安人寿不承担法律责任。

**合规要点**：
- 销售人员需明确告知客户，第三方服务供应商仅提供预约协助
- 客户与第三方之间的纠纷，引导客户直接与第三方沟通
- 服务提供方不对第三方的服务质量负责

### 条款3：就医参考信息仅供参考

服务提供的就医参考信息（如医院推荐、医生介绍）仅供参考，客户拥有最终决定权。客户应自行判断并选择适合的医疗机构和医生。

**合规要点**：
- 销售人员不得承诺"推荐的一定是最好的"
- 客户有权拒绝服务提供方的推荐
- 就医参考信息不作为医疗建议

### 条款4：客户应寻求并依赖医疗机构及医生的专业判断

服务提供方不提供医疗诊断或治疗建议。客户应寻求并依赖医疗机构及医生的专业判断，服务提供方不对医疗结果负责。

**合规要点**：
- 销售人员不得提供医疗建议
- 客户需自行与医生沟通病情
- 服务提供方不对医疗结果负责

### 条款5：就医产生的所有医疗费用由客户自行承担

服务仅提供预约、咨询等协助服务，不包含任何医疗费用。客户需自行承担挂号费、检查费、药费、住院费等所有医疗相关费用。

**合规要点**：
- 销售人员需明确告知客户医疗费用自理
- 名医大咖服务需客户自行付费购买
- 数字化管理的智能设备需客户自行付费购买

### 条款6：使用服务前需完成信息授权

客户需在平安金管家APP上完成个人信息授权，包括但不限于健康信息、就医记录的授权。未完成授权的客户无法使用服务。

**合规要点**：
- 信息授权是服务启动的前置条件
- 授权范围需明确告知客户
- 客户有权撤回授权

### 条款7：平安人寿有权调整服务内容

平安人寿有权根据业务需要调整服务内容、服务提供方、服务流程等，并会通过适当方式通知客户。调整后的服务内容以最新版本为准。

**合规要点**：
- 销售人员不得承诺服务内容永久不变
- 服务内容调整后，需及时通知客户
- 客户有权了解调整后的服务内容

### 条款8：名医大咖、数字化管理的智能设备需客户自行付费购买

名医大咖服务和数字化管理所需的智能设备（如血压计、血糖仪等）不在服务包内，客户需自行付费购买。

**合规要点**：
- 销售人员需明确告知客户名医大咖服务需自费
- 智能设备的价格和购买方式需告知客户
- 自费项目不享受服务包内的次数限制

## 合规操作指引

### 销售环节
1. 向客户出示本合规声明，并确保客户理解
2. 不得使用"保证治愈""包治百病"等绝对化用语
3. 不得将服务与保险理赔挂钩
4. 明确告知客户医疗费用自理
5. 明确告知客户名医大咖和智能设备需自费

### 服务使用环节
1. 客户使用服务前，需在APP上完成信息授权
2. 服务过程中，提醒客户就医参考信息仅供参考
3. 客户需自行与医疗机构确认费用和诊疗方案
4. 客户需自行判断并选择适合的医疗机构和医生

### 投诉处理环节
1. 客户对服务不满时，引导客户通过95511或4001666333投诉
2. 涉及医疗纠纷的，建议客户直接与医疗机构沟通
3. 服务内容调整引发的投诉，需耐心解释调整原因
4. 客户对第三方服务供应商不满时，引导客户直接与第三方沟通

## 知识缺口

- 服务内容调整的具体通知方式（短信/APP推送/公告）未明确

- 客户未完成信息授权时的替代方案未说明
- 服务内容调整后，客户是否有权终止服务未明确
- 客户撤回信息授权的流程和后果未说明
