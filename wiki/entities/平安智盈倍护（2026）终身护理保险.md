# 平安智盈倍护（2026）终身护理保险

## 产品定位

平安智盈倍护（2026）终身护理保险是一款终身护理保险产品，提供**疾病身故保险金**和**护理保险金**双重保障。产品核心定位是为客户提供因特定疾病或意外导致的长期护理状态下的经济补偿，同时兼顾疾病身故保障。

## 基础规则

| 规则项 | 内容 |
| :--- | :--- |
| 险种代码 | 2648 |
| 监管备案号 | 平安人寿〔2025〕护理保险 147号 |
| 投保年龄 | 0周岁（出生满28日）至64周岁 |
| 保险期间 | 终身 |
| 等待期 | 90日（意外伤害无等待期） |
| 犹豫期 | 20日 |
| 宽限期 | 60日 |
| 交费方式 | 分期支付 |

## 核心保障

### 疾病身故保险金

- **给付条件**：因疾病导致身故
- **给付金额**：max(所交保险费, 身故时现金价值)
- **给付后**：本合同终止

### 护理保险金

- **给付条件**（满足其一即可）：
  1. 因意外伤害原因或等待期满后因意外伤害之外的其他原因，确诊10种特定疾病之一
  2. 因意外伤害导致1-3级伤残（180日内）
- **给付金额**（分年龄段）：

| 年龄段 | 给付金额 |
| :--- | :--- |
| 18周岁前 | max(所交保险费, 现金价值) |
| 18周岁至61周岁 | max(所交保险费×160%, 现金价值) |
| 61周岁后 | max(所交保险费×120%, 现金价值×120%, 基本保额) |

- **责任互斥**：疾病身故保险金与护理保险金仅给付一项且以一次为限。

## 特定疾病列表（10种）

1. 严重特发性肺动脉高压
2. 严重脑中风后遗症
3. 严重脑炎后遗症或严重脑膜炎后遗症
4. 瘫痪
5. 严重阿尔茨海默病
6. 严重脑损伤
7. 严重原发性帕金森病
8. 严重运动神经元病
9. 双目失明
10. 多个肢体缺失

## 责任免除

### 疾病身故免责
- 吸毒、战争、核风险

### 护理保险金免责（非意外原因）
- 投保人故意杀害、被保险人故意犯罪/自伤/2年内自杀、吸毒、艾滋病、战争、核风险、遗传性疾病

### 护理保险金免责（意外原因）
- 投保人故意杀害、被保险人故意自伤/犯罪/自杀、吸毒、酒驾、战争、核风险、妊娠、精神障碍、药物过敏、未遵医嘱用药、高风险运动

## 其他重要权益

| 权益 | 规则 |
| :--- | :--- |
| 保单贷款 | 可贷现金价值80%，最长6个月 |
| 减保 | 同一保单年度内累计减少的基本保额之和不得超过生效时基本保额的20% |
| 保险事故通知 | 10日内通知 |
| 保险金核定 | 5日内核定，情形复杂30日内 |
| 宣告死亡 | 按法院宣告死亡判决处理，若重新出现需退还保险金 |
| 税收优惠 | 适用商业健康保险税收优惠政策的纳税人可享受个税优惠 |

## 合规提醒

- 销售时需明确告知客户责任免除条款。
- 护理保险金等待期为90日，意外伤害无等待期。
- 疾病身故保险金与护理保险金互斥，仅给付一项。
- 减保有20%年度上限。
- 保单贷款有80%现金价值上限和6个月期限。
