# 基本保险金额与费率计算

## 定义

基本保险金额是平安盛世优享（2026）养老年金保险中，**每万元趸交或年交保费**对应的保险金额，单位为人民币元。这是计算实际保险金的基础数值。

## 计算公式

```
实际基本保险金额 = 费率表对应值 × (实际缴纳保费 / 10,000)
```

**示例**：男性30岁，选择趸交10,000元，60岁开始领取，费率表对应值为96,018元，则基本保险金额为96,018元。

## 影响因素

| 因素 | 影响方向 | 说明 |
|------|----------|------|
| 性别 | 女性 > 男性 | 女性平均寿命更长，相同保费下获得更高保险金额 |
| 投保年龄 | 年龄越小 → 金额越高 | 越早投保，资金积累时间越长 |
| 开始领取年龄 | 领取越晚 → 金额越高 | 延迟领取，积累更多 |
| 缴费方式 | 缴费期越长 → 金额越高 | 长期缴费积累更多资金 |
| 领取方式 | 一次性领取 > 20年领取 | 一次性领取金额更高，20年领取提供稳定现金流 |
